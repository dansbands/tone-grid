# Migration inventory

This file tracks premium/pro concepts that were removed from or deferred out of ToneGrid so they can be migrated later.

## Removed or deferred from ToneGrid

### ToneSmith candidates
- structured preset categories beyond simple demo cycles
- advanced conditioning workflows
- neighbor-tone beating workflows as productized routines
- interval/drone routine systems
- custom routines
- saved sessions and recall
- guided conditioning flows

### ToneSmith Pro candidates
- resonance targeting
- comparison tooling
- luthier and shop workflows
- analysis and measurement features
- scientific adjustment tooling

## Shared core candidates
- reusable note utilities
- reusable instrument schema primitives
- reusable low-level audio helpers

## ToneGrid rule of thumb

If a feature feels like guidance, intelligence, analysis, professional workflow support, or long-term recall, it probably does not belong in ToneGrid.
